# ThermalComponents

This is the documentation for the `ThermalComponents` library.  Here you will find the documentation for the various definitions contained in `ThermalComponents`.

Note that this documentation is automatically generated primarily from the doc strings and metadata associated with those definitions.

## Types

  * `ConvectiveElement1D` - This partial model contains the basic connectors and variables to allow heat
  * `Element1D` - This partial model contains the basic connectors and variables to allow heat transfer models to be created that do not

## Components

  * `BodyRadiation` - Lumped thermal element for radiation heat transfer
  * `Convection` - Lumped thermal element for heat convection
  * `ConvectiveResistance` - Lumped thermal element for heat convection with resistance as an input.
  * `FixedHeatFlow` - Fixed heat flow boundary condition.
  * `FixedTemperature` - Fixed temperature boundary condition in kelvin.
  * `HeatCapacitor` - Lumped thermal element storing heat
  * `HeatFlowSensor` - Heat flow rate sensor
  * `HeatSystemTest` - Heat system with a fixed temperature source
  * `PrescribedHeatFlow` - Prescribed heat flow boundary condition.
  * `PrescribedTemperature` - This model represents a variable temperature boundary condition.
  * `RelativeTemperatureSensor` - Relative temperature sensor
  * `TemperatureOfTwoMassesTest` - This model connects 2 masses at different temperature. And, compares and tests
  * `TemperatureSensor` - Absolute temperature sensor
  * `ThermalConductor` - Lumped thermal element transporting heat without storing it
  * `ThermalResistor` - Lumped thermal element, with resistance, transporting heat without storing it.
